Project Name
    Dog Pictures API Project

Project Description
    This project consumes from an API and displays its retrieval on the page.

How to run the project
    1. Download the project
    2. Run the index.html file

How to use the project
    After opening the index.html file you will be displayed an image of a dog.
    If you want a new image you can click the "Get New Doggo" button indefinitely.
